# MDX allows you to use JSX components directly in your markdown content. This means you can make your content more interactive and dynamic while keeping the simplicity of markdown syntax.

## Basic Markdown Features

Here are some basic markdown features you can use:

- **Bold Text**: Use double asterisks
- _Italic Text_: Use single asterisks
- `Inline Code`: Use backticks
- [Links](https://nextjs.org): Use square brackets and parentheses
